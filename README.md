# PRFNN Stack Boilerplate

The PRFNN is a preformace optimised, self-hostable, scalable and containerised full web application development stack. It is designed to offer cutting edge features without gimics, and offer seasoned and capable developers full control and agency over each layer of their application.
This is traditional web application at it's finest, with a fast and cached data layer alongside a lightweight and preformant API. What you may be compromising in development speed and time to market with this stack, you make up for in preformance, cost-efficency when scaled, and control.
